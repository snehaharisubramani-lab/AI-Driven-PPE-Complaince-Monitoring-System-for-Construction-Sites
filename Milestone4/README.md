# Milestone 4 - Power BI Compliance Dashboard

# Overview
Interactive Power BI dashboard built on violation_log.json
collected during real-time inference.

# Files
- PPE_Dashboard.pbix - Power BI dashboard file
- Dashboard_Full.png - Complete dashboard screenshot
- violation_log.json - Data source for dashboard

# Dashboard Visuals
- 5 KPI cards - violation counts per PPE type
- Bar chart - PPE violation frequency analysis
- Donut chart - violation distribution percentages
- Detection log table - image, violation, status
- Slicers - filter by violation type and status

# Key Insights
- no_helmet is most frequent violation (34.43%)
- no_vest is second (31.15%)
- Helmets and vests = 66% of all violations
- Boot violations least frequent (6.56%)
