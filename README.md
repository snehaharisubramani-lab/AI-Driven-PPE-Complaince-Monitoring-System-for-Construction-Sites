Problem Statement:
Construction sites involve hundreds of workers across multiple zones where PPE such as helmets, vests, goggles, and boots are mandatory. Manual inspection is limited and violations often go unnoticed, leading to accidents and legal risks.
This project builds a real-time AI-based CCTV monitoring system that automatically detects PPE violations and sends instant alerts to site supervisors.

 Milestones
1. Site Architecture Design
2. PPE Dataset Annotation & Model Training
3. Real-Time Inference & Logging
4. Power BI Compliance Dashboard
5. Safety Outcome Analysis
